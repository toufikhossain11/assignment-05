## 1. Difference Between var, let, and const

In JavaScript, var, let, and const are used to declare variables.

var is the older way of declaring variables. 

let and const was introduced in ES6. It has block scope.

## 2. Spread Operator (...)

The spread operator (...) is used to expand elements of an array or properties of an object.

## 3. Difference Between map(), filter(), and forEach()

map(), filter(), and forEach() are commonly used array methods in JavaScript.

map() is used when we want to transform every element of an array and create a new array from it.

filter() is used when we want to select specific elements from an array based on a condition.

forEach() is used when we want to perform an action for each element in an array. Unlike map() and filter(), it does not return a new array.

## 4. Arrow Function

Arrow functions are a modern and shorter way to write functions in JavaScript. They were introduced in ES6.

## 5. Template Literals

Template literals are a modern way of working with strings in JavaScript. They are written using backticks instead of single or double quotes.
